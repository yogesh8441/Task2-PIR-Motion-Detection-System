# Motion Detection Security System Using Arduino UNO and PIR Sensor

## Overview
This project implements a motion detection system using Arduino UNO and a PIR sensor.

## Components Used
- Arduino UNO
- PIR Motion Sensor
- LED
- Wokwi Simulator

## Working
When motion is detected, the PIR sensor sends a HIGH signal to Arduino. Arduino turns ON the LED and displays "Motion Detected!" on the Serial Monitor.

## Results
- Motion Detected → LED ON
- No Motion → LED OFF
